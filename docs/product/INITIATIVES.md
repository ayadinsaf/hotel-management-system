# Initiatives

## I1 — Core system foundations
Build the shared backend that all three products depend on.
Microservices, databases, API gateway, CI/CD pipeline.

## I2 — Staff app
Give hotel staff a reliable tool to manage daily operations.
Covers reservations, room management, check-in/out, guest profiles.

## I3 — Resilience and scalability
Decouple services using event-driven architecture.
Introduce a message broker and async communication between services.

## I4 — Customer booking experience
Allow guests to search availability and book rooms online.
Covers the booking site and payment integration.

## I5 — Data and business intelligence
Give management visibility into business performance.
Data pipeline, Snowflake warehouse, dbt models, dashboards.
Lays the groundwork for dynamic pricing and demand forecasting.
